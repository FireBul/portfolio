# Wording Upgrade Audit (Expert POV)

## Objective
면접관/채용담당/의사결정자 관점에서 신뢰도와 전문성을 높이는 카피로 전면 정렬.

## Applied upgrades
- About/Projects/Contact/Leadership 메타 및 타이틀을 전문가형으로 재작성
- Projects 서브타이틀을 KPI 실행 프레임(문제정의→실행→성과)으로 고정
- Contact 카피를 실행/응답 기준 중심으로 정리
- Home에 기술 스펙(SEO/CRM/Chatbot/Analytics/UX) 명시

## High-impact writing principles used
1. 결과 중심 문장 (성과 수치 우선)
2. 모호한 형용사 축소, 실행 동사 강화
3. 현재/과거 시점 불일치 제거 (인터파크 2025.12 퇴사 반영)
4. 채용 관점 정보 우선 배치

## Next wording pass (recommended)
- 모든 프로젝트 상세 페이지에 동일한 5블록 템플릿 적용:
  - Context / Problem / Action / KPI / Lesson
- 숫자 없는 문장은 가능하면 수치형 근거로 대체
- 영어 title slug와 한글 카피 균형 조정
