## 📐 상세한 이미지 규격 및 촬영 가이드

### 🖼️ 프로필 사진 (profile.jpg) - 최고 우선순위

**기술적 요구사항:**
- **크기**: 300x400px (세로 비율 3:4)
- **해상도**: 최소 300 DPI
- **용량**: 500KB 이하 (TinyPNG로 압축 후)
- **형식**: JPG 권장 (JPEG 고품질)
- **색상**: sRGB 컬러스페이스

**촬영 가이드:**
- **복장**: 정장 또는 비즈니스 캐주얼 (네이비, 차콜, 블랙 권장)
- **표정**: 자연스러운 미소, 신뢰감 있는 눈빛
- **배경**: 밝은 단색 배경 (흰색, 연한 회색)
- **조명**: 자연광 또는 스튜디오 조명 (그림자 최소화)
- **각도**: 정면 또는 약간 좌측/우측 (15도 이내)
- **구도**: 가슴 위부터 머리까지 (헤드샷)

- **구도**: 모든 멤버의 얼굴이 보이도록 촬영
- **분위기**: 프로페셜하면서도 따뜻한 분위기

**필수 포함 요소:**
- 바이올린 9명, 비올라 3명, 첼로 3명, 플루트 2명, 클라리넷 3명

### 🔍 이미지 품질 체크리스트

#### 기술적 품질
- [ ] **해상도**: 웹용 최소 72 DPI, 인쇄용 300 DPI
- [ ] **크기**: 권장 픽셀 크기 준수
- [ ] **용량**: 각 이미지 1MB 이하 (프로필은 500KB 이하)
- [ ] **형식**: 사진은 JPG, 그래픽은 PNG
- [ ] **색상**: sRGB 컬러스페이스 사용
- [ ] **압축**: TinyPNG로 최적화 완료

#### 시각적 품질
- [ ] **선명도**: 텍스트와 디테일이 명확히 보임
- [ ] **노이즈**: 불필요한 노이즈나 아티팩트 없음
- [ ] **색상**: 자연스럽고 일관된 색감
- [ ] **구도**: 주요 요소가 중앙에 잘 배치
- [ ] **조명**: 고른 조명으로 그림자 최소화
- [ ] **배경**: 깔끔하고 주제에 방해되지 않음

#### 콘텐츠 품질
- [ ] **관련성**: 해당 페이지/섹션과 직접적 연관
- [ ] **정확성**: 실제 데이터나 현실을 반영
- [ ] **전문성**: 비즈니스 환경에 적합한 수준
- [ ] **스토리텔링**: 명확한 메시지 전달
- [ ] **차별화**: 다른 포트폴리오와 구별되는 독창성
