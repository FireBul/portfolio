# Portfolio Blueprint 2026

## Objective
- Evolve the portfolio into an expert-grade product/AI/ML narrative site.
- Present selected private projects safely (no sensitive implementation leakage).
- Add measurable business credibility (CPC/CTR/click/contact metrics).

## Information Architecture
1. Home
   - Hero: role + domain + proof
   - KPI strip (impact metrics)
   - Featured case studies
   - R&D highlights (private-safe)
2. Projects
   - Filter by domain (Ads / Commerce / AI / ML / Bio)
   - Each card includes outcome metrics + role + stack
3. Project Detail
   - Problem → Approach → Execution → Outcome
   - Optional analytics panel (CPC, CTR, conversion, click count)
4. Contact (CRM-style)
   - Inquiry type, budget range, timeline, referral source
   - Track clicks/submits/events

## Feature Roadmap
- Phase 1: Messaging refresh + KPI framing + career update
- Phase 2: Private project section + structured case-study format
- Phase 3: Chatbot-style Q&A + analytics widgets
- Phase 4: CRM contact flow + event tracking dashboard

## Guardrails
- Keep private projects abstracted (architecture/impact level).
- Avoid leaking confidential data, source, or partner-sensitive details.
- Maintain polished, minimal UI with strong readability.
