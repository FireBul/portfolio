AURORA 정량 분석 딥다이브

96개 전략 × 22개 시장 구간 × 33,752건 거래의 체계적 분석

96전략 변형
22시장 구간
33,752분석 거래
52통계 분석
8앙상블 멤버

1. 분석 파이프라인 아키텍처

6단계 분석 파이프라인으로 52개 통계 분석을 43.7초 만에 실행합니다. 96개 전략 유니버스에서 최종 8-전략 앙상블까지 체계적 필터링을 수행합니다.

2. 전략 유니버스 탐색

96개 전략의 상관구조, 클러스터링, 생존편향을 다각도로 검증합니다. 계층적 클러스터링과 네트워크 분석으로 전략 간 중복성과 분산 투자 기회를 식별합니다.

3. 시장 레짐 & 안정성

13가지 시장 레짐에서의 성과와 전천후 안정성을 평가합니다. 레짐 전환 확률 행렬로 시장 상태 간 전이 패턴을 분석합니다.

4. 거래 수준 포렌식

33,752건 개별 거래의 손익 분포, 시간대, 보유기간, 연속 패턴을 분석합니다. 청산 사유 분포로 전략별 시그널 품질을 진단합니다.

5. 모델 & 시그널 품질 감사

과적합 검증, 확률 보정, 시그널 발화율, 특성 중요도를 감사합니다. 티어 시스템의 통계적 유의성을 Mann-Whitney U 검정으로 확인합니다.

6. 앙상블 설계 & 리스크 관리

최종 8-전략 앙상블의 성과, 위기 대응, 알파 감쇠, 수수료 민감도를 종합합니다. 22개 구간 전체에서 수익을 달성한 강건한 포트폴리오입니다.

7. 기술 스택 & 방법론

통계 검정
Mann-Whitney U, Wald-Wolfowitz, Spearman, 부트스트랩 CI
클러스터링
Ward 계층적 클러스터링, 덴드로그램, 상관 네트워크
리스크 관리
VaR/CVaR, 켈리 기준, 시나리오 분석, 스트레스 테스트
ML 파이프라인
XGBoost, CatBoost, LightGBM, HGB + 앙상블 voting
데이터 규모
631K 백테스트 행, 403K 시그널, 33K 거래
분석 엔진
Python + NumPy + SciPy + Plotly (52개 분석, 43.7초)
