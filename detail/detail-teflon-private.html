프로젝트 배경

반도체 제조 공정에서 사용되는 PFA/PTFE 튜브는 미세한 표면 결함도 공정 오염으로 이어질 수 있습니다. 기존 육안 검사는 작업자 피로도와 일관성 문제가 있었고, 단순한 모델 정확도가 아니라 현장에서 실제로 돌아가는 시스템이 필요했습니다.

핵심 인사이트: "현장은 모델만으로 돌아가지 않는다"
현상 카메라 소스 다양성(USB/IP/RTSP), 운영자 즉시 판단 어려운 UI, 업데이트 시 현장 중단 리스크
조치 HAL 추상화 + 멀티카메라 핫플러그 + 웹 기반 운영 UI + OTA 원격 업데이트
결과 13ms 추론, 14,551건/세션 검사 처리, 무중단 모델 업데이트 가능
