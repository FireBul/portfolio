프로젝트 배경

Pure-Flon은 반도체/의료/화학 공정용 PTFE 튜브를 제조하는 B2B 기업으로, 한국·일본·대만·중국 시장을 대상으로 합니다. 기존 웹사이트는 단일 언어, 정적 페이지로 견적 프로세스와 고객 관리가 분리되어 있었습니다.

핵심 과제
현상 단일 언어 정적 사이트, 견적은 이메일 수동 처리, 고객별 주문 이력 추적 불가
조치 4개 언어 지원 + 온라인 견적 시스템 + 고객 포털 + PWA + 엣지 배포
결과 동아시아 4개 시장 접근성 확보, 견적 프로세스 자동화, Core Web Vitals 최적화

PWA & 성능 최적화

캐시 전략 App Shell: Cache-first, 페이지: Network-first, 이미지: Cache-first, 오프라인 폴백
